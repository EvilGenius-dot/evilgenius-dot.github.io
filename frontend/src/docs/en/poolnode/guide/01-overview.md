# PoolNode Documentation

PoolNode is the real-pool layer used with RustMinerSystem when an operator wants settlement-level fee control instead of only forwarding traffic to a third-party pool.

## Scope

- Define the PoolNode role in a RustMinerSystem deployment.
- Plan node connectivity, settlement configuration, and recovery procedures.
- Connect the PoolNode mobile app as the user-facing asset, worker, and earnings entry.
- Keep pool-node operations separate from the core RustMinerSystem proxy documentation.

## Suggested Document Structure

Start with an overview, then add pages for node setup, settlement rules, mobile app integration, monitoring, upgrades, and incident recovery.
