# RMS Documentation

RMS is the optional secure client deployed near miners when a site needs encrypted local compression, fewer public connections, or a controlled access path into RustMinerSystem.

## Scope

- Explain when RMS is useful and when direct miner access is enough.
- Record client deployment, pairing, version compatibility, and rollback steps.
- Separate local network optimization guidance from RustMinerSystem server configuration.
- Keep future platform-specific installation pages under the RMS document collection.
