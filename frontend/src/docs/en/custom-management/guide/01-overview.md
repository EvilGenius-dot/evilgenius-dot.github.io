# Custom Management Documentation

Custom management covers the operational workflow around branded RustMiner builds, customer access, hosted deployment, release updates, and support handoff.

## Scope

- Plan the information needed for a branded build, including name, logo, footer content, and promotional pages.
- Manage customer-facing access, permissions, and delivery status.
- Track hosted deployment, one-click install links, and update channels.
- Separate commercial customization workflows from core RustMinerSystem server documentation.

## Suggested Document Structure

Start with this overview, then add pages for customer onboarding, brand asset preparation, build generation, hosted deployment, hot updates, permissions, and support procedures.
