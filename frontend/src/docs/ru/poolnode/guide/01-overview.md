# Документация PoolNode

PoolNode является слоем настоящего пула в связке с RustMinerSystem, когда оператору нужен контроль комиссии на уровне расчета монет, а не только пересылка трафика в сторонний пул.

## Область документации

- Определить роль PoolNode в развертывании RustMinerSystem.
- Спланировать сетевую связность, настройки расчетов и процедуры восстановления.
- Подключить мобильное приложение PoolNode как вход для активов, воркеров и доходов.
- Отделить эксплуатацию pool-node от документации core proxy RustMinerSystem.

Начните с обзора, затем добавьте страницы по настройке узла, правилам расчетов, интеграции мобильного приложения, мониторингу, обновлениям и восстановлению после инцидентов.
