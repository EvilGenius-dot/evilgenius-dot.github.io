# Документация Custom Management

Custom Management описывает рабочий процесс вокруг брендированных сборок RustMiner, клиентского доступа, hosted-развертывания, обновлений релизов и передачи в поддержку.

## Область документации

- Подготовить данные для брендированной сборки: имя программы, логотип, footer-информацию и промо-страницы.
- Управлять клиентским доступом, правами и статусом поставки.
- Отслеживать hosted-развертывание, one-click install ссылки и каналы обновлений.
- Отделить коммерческую кастомизацию от основной серверной документации RustMinerSystem.

## Рекомендуемая структура

Начните с обзора, затем добавьте страницы по onboarding клиента, подготовке бренд-материалов, генерации сборки, hosted-развертыванию, hot update, правам и support-процессам.
