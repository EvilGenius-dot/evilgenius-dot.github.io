# PoolNode 文档

PoolNode 是 RustMinerSystem 真矿池模式中的节点层，适合需要在虚拟币结算层面控制费率，而不是只做第三方矿池转发的运营方。

## 文档范围

- 明确 PoolNode 在 RustMinerSystem 部署中的职责边界。
- 规划节点连通性、结算配置和故障恢复流程。
- 对接 PoolNode 移动应用，作为用户查看资产、矿工和收益的入口。
- 将真矿池节点运维从 RustMinerSystem 核心 Proxy 文档中独立出来。

## 建议文档结构

先保留概览页，再逐步补充节点部署、结算规则、移动端接入、监控、升级和事故恢复等独立页面。
