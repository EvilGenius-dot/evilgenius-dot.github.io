# 定制管理文档

定制管理用于记录品牌定制版 RustMiner 的交付和运维流程，包括客户资料、品牌素材、托管部署、版本更新和售后支持。

## 文档范围

- 规划定制版本所需资料，例如程序名称、LOGO、footer 信息和推广页面内容。
- 管理客户访问入口、权限范围和交付状态。
- 跟踪托管部署、一键安装链接和更新通道。
- 将商业定制流程从 RustMinerSystem 核心服务端文档中独立出来。

## 建议文档结构

先保留概览页，后续可以继续补充客户开通、品牌素材准备、定制构建生成、托管部署、热更新、权限管理和支持流程等页面。
