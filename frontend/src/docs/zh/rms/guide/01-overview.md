# RMS 文档

RMS 是可选安全客户端，通常部署在矿场本地，用于加密压缩矿机流量、减少公网连接数，并把矿机以受控链路接入 RustMinerSystem。

## 文档范围

- 说明什么时候应该使用 RMS，什么时候可以直接让矿机接入。
- 记录客户端部署、配对、版本兼容性和回滚步骤。
- 将本地网络优化内容从 RustMinerSystem 服务端配置中拆出来。
- 后续按平台补充 Linux、Windows 等安装页面。

## 建议文档结构

建议继续拆分 Linux 与 Windows 部署、服务端配对、网络容量评估、升级策略、诊断和紧急旁路等页面。
