# 运维

## 日常检查

- 检查服务运行状态。
- 检查连接数、错误日志和延迟。
- 检查上游矿池或节点是否稳定。
- 检查系统资源占用。

## 升级流程

1. 阅读发布说明。
2. 备份当前配置文件。
3. 在测试环境验证新版本。
4. 安排维护窗口。
5. 升级生产节点。
6. 验证连接、日志和矿机提交状态。

## 故障排查

### 服务无法启动

- 检查配置文件格式。
- 检查端口是否被占用。
- 检查当前用户是否具备启动权限。

### 无法连接矿池

- 检查网络连通性。
- 检查矿池地址和端口。
- 检查账号、矿工名和密码。

## FAQ

### RustMinerSystem 可以同时支持多少矿机？

请根据服务器配置、网络带宽、上游矿池稳定性和实际压测结果确定。正式上线前建议进行分批接入测试。

### 升级时是否需要停机？

取决于部署方式和版本变更内容。生产环境建议安排维护窗口，并提前准备回滚方案。
