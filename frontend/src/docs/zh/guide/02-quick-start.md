# 快速开始

## 环境要求

| 项目 | 要求 |
| --- | --- |
| 操作系统 | Windows / Linux / ARM / ARMV7 |
| 网络 | 可访问目标矿池或自建节点 |
| 权限 | 具备安装、配置和启动服务的权限 |

## 基础流程

1. 下载对应平台的 RustMinerSystem 服务端包。
2. 配置矿池地址、端口和矿工凭据。
3. 启动服务并检查日志。
4. 接入少量矿机进行验证。
5. 验证稳定后逐步接入生产负载。
