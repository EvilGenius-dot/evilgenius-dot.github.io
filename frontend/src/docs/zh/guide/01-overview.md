# RustMinerSystem 文档

> 本文档用于记录 RustMinerSystem 的安装、配置、运行和运维流程。请将示例中的占位内容替换为你的真实部署信息。

## 概览

RustMinerSystem 可以作为传统矿池代理使用，也可以用于搭建自有矿池节点。配套 RMS 客户端可对矿机通信数据进行加密和压缩，帮助降低带宽压力并提升传输安全性。

### 适用场景

- 矿场需要统一管理矿机接入路径。
- 运营方需要部署自有矿池节点。
- 网络环境需要降低带宽占用或增强传输安全。
- 团队需要可复制的安装、配置和升级流程。

## 运行模式

### 传统矿池代理

该模式适用于将矿机流量转发到现有矿池的场景。建议先使用测试矿机验证连接稳定性，再扩大接入规模。

### 自建矿池节点

该模式适用于需要更强控制能力的运营场景。部署前请确认节点同步状态、网络连通性和故障恢复流程。

### RMS 安全客户端

RMS 客户端用于加密和压缩矿机通信数据。启用前请确认服务端和客户端版本兼容，并记录回滚方案。
