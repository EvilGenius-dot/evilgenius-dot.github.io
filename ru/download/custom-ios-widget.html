Custom iOS widget

Платные custom-версии RustMinerSystem могут использовать этот iOS widget на базе Scriptable, чтобы прямо на домашнем экране iPhone видеть online-устройства, ожидание проверки, число клиентов, устройства по монетам, хешрейт и тренды.

Для клиентов с уже подключенной платной custom-версией RustMinerSystem: online-устройства, ожидание проверки, число клиентов, top монет по устройствам, хешрейт и короткие тренды.

Генератор персонального скрипта

Введите CTL Token, чтобы скачать готовый Scriptable-скрипт

Страница вставляет вручную созданный CTL Token в Scriptable-скрипт и создает файл локально в браузере. Token не сохраняется в коде сайта и не отправляется на сервер.

Сгенерировать персональный скрипт

Вставьте CTL Token, созданный для клиента. Страница запишет его в переменную CTL_TOKEN и скачает новый JS-файл.

Token используется только для этой загрузки в браузере. Не создавайте скрипты на публичных устройствах.

Безопасность token

CTL Token является ключом, через который widget читает статистику custom-версии. Передавайте его только соответствующему клиенту.

  • Страница генерирует скрипт только в текущем браузере. Token не попадает в репозиторий и не загружается на сервер.
  • Если token раскрыт, создайте новый token и попросите клиента скачать скрипт заново.
  • Для обновления статистики iPhone должен иметь доступ к API custom management.
Генератор персонального скрипта

Что показывает widget

Скрипт обращается к API custom management и сжимает основные данные под формат iOS widget.

Ключевые числа

Верхняя строка показывает online-устройства, ожидание проверки и число клиентов.

Список монет

Widget показывает top 5 монет по числу устройств, хешрейт и мини-графики тренда.

Автообновление

Скрипт планирует обновление через 15 минут, что удобно для быстрых проверок с домашнего экрана.

Процесс установки

После получения скрипта клиенту нужно только импортировать его в Scriptable и привязать к iOS widget.

01

Установите Scriptable

Найдите Scriptable в App Store на iPhone, установите приложение и разрешите сетевой доступ при первом запуске.

02

Создайте CTL Token

Оператор custom-сервиса вручную создает token для клиента и проверяет, что он соответствует нужным данным custom-версии.

03

Скачайте персональный скрипт

Введите CTL Token на этой странице, сгенерируйте загрузку и получите JavaScript-файл с уже вставленным token.

04

Импортируйте в Scriptable

Отправьте JS-файл на iPhone и выберите Scriptable в share menu, либо создайте новый скрипт в Scriptable и вставьте содержимое файла.

05

Добавьте widget на экран

Удерживайте домашний экран iPhone, нажмите plus, выберите большой Scriptable widget, затем отредактируйте widget и выберите импортированный скрипт.
