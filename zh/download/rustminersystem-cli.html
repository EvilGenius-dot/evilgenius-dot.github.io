RustMinerSystem CLI

RustMinerSystem CLI 已集成到 RustMinerSystem 主安装脚本中，面向不想或不方便使用 Web 管理后台的用户。运行同一条 Linux 安装命令后，在脚本菜单中选择命令行工具即可安装。

阅读 CLI 文档
rustminer把 RustMinerSystem 管理入口带到终端里

面向服务器终端、跳板机和安全隔离环境。安装完成后可直接运行 rustminer 打开 CLI，也可用 rustminer start、stop、restart 控制服务。

命令行工具下载

使用同一条安装脚本启用 CLI 模式

RustMinerSystem CLI 通过主安装脚本提供安装入口。先在服务器上完成 RustMinerSystem 部署，再选择菜单中的命令行工具，脚本会下载 CLI、创建 rustminer 命令启动器，并把系统切换到 WEB 访问关闭的 CLI 使用模式。

一条命令进入安装菜单

CLI 与 RustMinerSystem 共用主安装脚本。请使用 root 用户在 Linux 服务器执行命令，选择语言后按菜单提示继续。

RustMinerSystem / CLI 安装命令
bash <(curl -s -L https://github.com/EvilGenius-dot/RustMinerSystem/raw/main/install.sh)

如果服务器尚未安装 RustMinerSystem，请先选择 1 安装主程序；随后选择 22 安装命令行工具。安装 CLI 前脚本会提示关闭 WEB 访问，确认后会自动切换到 CLI 模式。

脚本中的 CLI 相关菜单

同一个安装脚本同时负责主程序、WEB 访问开关、运行状态和 CLI 工具管理。

  • 22安装命令行工具，并创建 /usr/local/bin/rustminer 启动器。
  • 23卸载命令行工具，清理脚本管理的 rustminer 启动器和 CLI 文件。
  • 10/11开启或关闭 WEB 访问；CLI 模式要求 WEB 访问处于关闭状态。
  • 21查看 RustMinerSystem 运行状态、端口、CLI 工具和路径信息。

安装后的常用命令

安装脚本会下载 RustMinerSystemCLI-0.1.0，并生成 rustminer 命令。无参数运行进入 CLI，带参数可控制主服务。

$ rustminer$ rustminer start$ rustminer stop$ rustminer restart

终端入口

无参数运行 rustminer 即可打开 CLI 工具，适合只通过服务器终端查看统计和执行操作的场景。

关闭 WEB 访问

脚本安装 CLI 时会将 ENABLE_WEB_UI 设置为 0，降低后台页面暴露面，统计数据改由 CLI 查看。

服务控制

rustminer start、rustminer stop、rustminer restart 可启动、停止或重启 RustMinerSystem，方便基础运维。
